# Sarah Keno

A minimal Vite + React + TypeScript + Tailwind CSS starter for a multi-card Keno game UI shell.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Current UI shell includes

- Header with game title and bankroll display
- Controls panel: number of cards (1–8), bet per card, Draw button (disabled until valid)
- Cards grid area with placeholder cards and a number picker (1–80)
- Drawn numbers area (empty state)
- History panel (empty state)
