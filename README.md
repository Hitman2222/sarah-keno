# Sarah Keno

A minimal Vite + React + TypeScript + Tailwind CSS Keno project.

## Install

```bash
npm install
```

## Run dev server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Run tests

```bash
npm run test
```

## Preview build

```bash
npm run preview
```

## Current UI shell + gameplay

- Header with title and bankroll
- Controls for cards (1–8), bet per card, payout preset, and Draw
- Per-card number picker (1–80), hits, wins, and daub highlighting
- Quick Pick All, Clear All, Repeat Last Round
- Drawn numbers panel
- Last 20 rounds history panel
